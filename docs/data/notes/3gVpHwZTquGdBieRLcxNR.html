Conventional Commits

#summary (Private)

  • Commits MUST be prefixed with a type, which consists of a noun

    • e.g feat:, fix:, build:, chore:, ci:, docs:, style:, refactor:, perf:, test:

  • scope after type is optional

  • ! for breaking change

  • p.hasURL (Private) https://www.conventionalcommits.org

  • p.isRelated (Private) https://gist.github.com/joshbuchea/6f47e86d2510bce28f8e7f42ae84c716

  • p.hasSpecification (Private) https://www.conventionalcommits.org/en/v1.0.0/
