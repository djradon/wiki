Standards

The Web standards implemented by Trellis include:

  • RFC 7089 (HTTP Framework for Time-Based Access to Resource States
  • RFC 7519 (JSON Web Tokens)
  • SOLID WebAC (Authorization and Access Control)
  • W3C Activity Streams
  • W3C JSON-LD
  • W3C Linked Data Notifications
  • W3C Linked Data Platform
  • W3C RDF 1.1 Concepts and Abstract Syntax
  • W3C Web Annotation Protocol
  • W3C WebID
  • W3C WebSub WebSub
