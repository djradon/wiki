## Support

### RDF language support:

-   SPARQL (`.rq`)
-   Turtle (`.ttl`)
-   Trig (`.trig`)
-   SHACL (`.shacl`)

### Stardog language support:

-   Stardog Mapping Syntax 2 (`.sms`, `.sms2`)
-   Stardog Rules Syntax (`.srs`)
