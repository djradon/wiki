Upper Ontology

  • #definition (Private): identifies those generic types of entities which belong to the formal structure of the world (e.g., OBJECT, PROCESS , SPATIAL REGION ), together with formal specifications of how those types of entities are related to others (e.g., OBJECT participates_in PROCESS ).
