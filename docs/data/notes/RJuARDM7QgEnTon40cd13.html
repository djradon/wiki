GraphDB

  • Graph Database
  • instanceOf Graph Database
  • written-in: java
  • supports RDF-star Kafka
  • produced_by: Ontotext
  • based_on: Rdf4j
  • related: Ontotext Platform

Features

  • embeddable
  • Synchronize data to downstream systems with the Kafka connector.
  • can be used with Jena, see https://graphdb.ontotext.com/documentation/9.1/free/using-graphdb-with-jena.html
  • SPARQL-MM supports "system time" queries
    • "Note that querying the history log may be slow for big history logs. This is why we recommend using filters to reduce the number of history entries if you have a big repository."
  • GeoSPARQL
  • javascript support allows custom functions
    • could be useful in context of auto-versioning Star Vers Versioning and Timestamping Rdf Data by Means of Rdf Star an Approach Based on Annotated Triples
  • spatial extensions
