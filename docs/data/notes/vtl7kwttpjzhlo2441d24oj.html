Planet Explorers

  • url https://planetexplorers.pathea.net/
  • description Planet Explorers is an open world voxel based sandbox adventure rpg game set on a distant planet. The game uses a new OpenCL system based on the Unity 3D engine to allow players to change the terrain in anyway, create new objects in any form, and do it anywhere.
