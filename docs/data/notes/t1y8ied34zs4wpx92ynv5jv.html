Issues

  • if we add the qualifiers for both statements, then we would not know which start date pairs with which end date, for example.
    • Solution A proposed workaround involves adding intermediate nodes to denote different occurrences of quoted triples, but this requires a reserved term 
      - @dean-allemang might say: "A held position/office isn’t some new thing I am inventing just to get around some deficiency in my modeling system; it’s actually the way people talk about holding public office."
